## Crypto Arbitrage Tools Deep Dive

### ArbitrageScanner.io (Premium Reference)
**Types of Arbitrage:**
- Spot-to-spot (CEX-CEX): Buy low on Exchange A, sell high on Exchange B
- Spot-futures: Long spot + short futures hedge
- Futures-to-futures: Cross-exchange perpetual contract spreads
- CEX-DEX: Centralized vs decentralized exchange spreads

**Premium Features Users Pay For:**
- Real-time scanning across 50+ CEX and 25+ DEX (40 blockchains)
- On-chain wallet analysis (earnings, win rate, P&L)
- AI-powered similar wallet discovery (272 analytical criteria)
- Mass whale wallet tracking
- Real-time Telegram notifications
- Funding rate tracking for perpetual contracts
- Market sentiment tracking (Telegram channels + Reddit)
- Personal account manager
- 70+ documented profitable use cases

**Key Features of a Good Arbitrage Calculator:**
1. Real-time price comparison across exchanges
2. Transaction fee + withdrawal fee deduction
3. Network fee consideration (different chains have different costs)
4. Slippage estimation
5. Transfer time estimation (opportunity may vanish)
6. Profit display in absolute and percentage terms
7. Custom capital input
8. Exchange API integration
9. Risk controls (max trade amounts, stop-loss)
10. Matching withdrawal networks
